# 上海青少年AI教育情报 2026-05-28

## 今日结论

1. 上海公开市场里，“英语机构转综合素养”还在继续，但真正被高频包装出来的不是“AI+财商+英语表达”三合一，而是“英语/语言基础 + 科创/机器人 + 项目制展示 + 营地化体验”。
2. 头部或老牌机构更像在做“横向扩品”而不是“纵深重做教学法”：英孚强调全人教育与机器人玩创，爱贝把“英语筑基”延展到“AI智学+全科”，昂立则直接把语言、人文、逻辑、托管、科创和PBL营打成一站式。
3. 今天最值得注意的新型组合，不是面向低龄儿童的成熟商业课，而是高校/国际化项目把“AI + 创业/商业模拟 + 英语展示”做成高客单夏校或体验营，例如港大上海科创与创业体验营、上纽大全英文AI暑期项目、包校暑期“Young Entrepreneur”方向。说明高端家庭愿意为“AI+商业表达”付费，但低龄民办教培端尚未形成清晰头部。
4. 家长转化侧的高频词已经很稳定：全英文沉浸、项目制、真实问题、成果展示、名校/专家背书、竞赛/证书、跨学科、未来素养。单纯讲“学AI工具”不够，必须绑定“作品、展示、升学想象”。
5. 对樱桃图书馆/登顶升学最重要的机会仍然是白地切入：把低龄可懂的商业情境、AI工具实操、英语表达输出、家长可见成果展示，压缩成半日营/五日营/长期班连续转化链，而不是跟着做泛机器人或泛编程。

## 英语机构转型观察

- 英孚公开前台已经不再只卖英语，官网把“全人教育”“机器人玩创”“跨文化互动表演”“国潮小匠人”并列陈列，说明其包装逻辑是语言做底座，素养和场景做提价与续费。来源：<https://www.yingfu.com.cn/>、<https://www.yingfu.com.cn/city/>
- 爱贝的转型更直接。其新站点“爱贝Ai智学中心”明确写“英语筑基+全科拓展”的 OMO 学习生态，覆盖小学到高中，属于从单科英语往学习机/智学中心/全科服务延展。来源：<https://www.abieai.com/index.html>
- 昂立的动作最像“综合素养超市”。官网把3-12岁一站式服务、12-18岁青少年成长中心、PBL实践应用创造营、Hello科学、研游学并排展示，语言不再是唯一主轴，而是综合能力的一部分。来源：<https://www.onlyedu.com/>
- 瑞思今天没有检索到上海本地、可信、在售的新AI/财商/英语融合官方页，公开前台教育声量偏弱。这本身也是信号：老英语品牌并非都成功转入新叙事。
- 结论不是“英语赛道结束”，而是“纯英语低故事性产品越来越难单独讲价”。真正还在前台抢眼球的，是英语与科技、项目、舞台、营地、升学想象的组合。

## AI+财商+英语表达专项观察

- 今天仍未发现上海本地少儿教培市场里，已经成熟跑通并高频招生的“AI + 财商 + 英语表达”标准化产品。公开可验证案例更多停留在两两组合。
- 可验证的近邻案例主要有三类：
  1. `AI + 英语表达`：上纽大AI暑期项目、新兴技术酷夏营都把全英文授课、项目展示、国际化沟通能力写进核心卖点。来源：<https://www.shnyu.edu.cn/cn/academics/summer-academy/ai>、<https://shanghai.nyu.edu/cn/academics/summer-academy/ima-emerging-technologies>
  2. `AI + 创业/商业表达`：港大上海×未来论坛体验营把 AI 时代准备、商业挑战、快速原型、Mini Expo、最终 pitch 连成闭环；临港 Future Maker Camp 也把创业模拟、roadshow、跨文化表达放在主流程。来源：<https://www.summerinstitute.hku.hk/programmes/detail/siss2618>、<https://english.shanghai.gov.cn/en-Latest-TalentsinShanghai/20260509/9f771cb0d35a4feeb54a543a6c5975a0.html>
  3. `商业启蒙 + AI工具 + 双语输出`：包校 2026 夏校公开资料中出现 “Young Entrepreneur: Cracking the Code of Business”，强调学生扮演“Startup CEO”，使用 AI 做市场分析、宣传物料、文案和商业路演，且为双语授课。这个方向与我方最接近，但目前更像国际学校暑期项目而非大众教培常规班。来源：<https://www.ykpaoschool.cn/Public/Uploads/ueditor/upload/file/20260325/1774424911398671.pdf>（单源线索，来自公开PDF摘要）
- 关于 `财商` 本身，公开市场仍多为单独财商启蒙、桌游、消费观教育，和AI或英语表达的整合度不高。今天没有检索到上海头部机构把“儿童财商 + AI工具 + 英语输出”做成强品牌主品。

## 高频卖点/内容趋势

- 高频卖点1：`项目制/PBL`。昂立、政府科普课程、国际夏校都在强调真实问题、团队协作、成果展示。

## 对樱桃图书馆/登顶升学的启发

1. 今天市场上最值得借鉴的 1-2 个动作
   - 借鉴动作一：把“AI学习”改写成“有角色的项目任务”。港大和包校都不是先讲工具，而是先给“创业者/CEO/项目团队”身份，再让AI服务于研究、原型、展示。
   - 借鉴动作二：把“英语能力”藏进结果交付，而不是独立售卖。上纽大和新兴技术酷夏营的核心不是口语课，而是英语环境下完成项目、展示作品、与导师交流。

2. 今天市场上最不该跟的 1 个方向
   - 不该跟“泛AI启蒙+泛编程+泛证书”空包装。如果没有真实项目情境和家长可见成果，很容易沦为和现有科创/编程机构正面同质化。

3. 对“AI时代小小CEO体验日”的启发
   - 最适合做成 `一个真实消费/品牌问题 + 一个AI工具 + 一次英文展示`。例如“帮一家儿童书店设计六一快闪摊位”：孩子用AI做消费者画像、海报文案、摊位命名，最后做60秒英文电梯演讲。体验日的关键不是知识量，而是家长现场就能看见孩子“像个小老板在表达”。

4. 对“AI时代小小CEO五日营”的启发
   - 五日营适合走完整商业闭环：发现问题、用户采访、产品构想、预算与定价、AI生成物料、英文路演。市场上已有 AI+创业 camp，但低龄段、亲子可见、中文商业理解与英文输出并重的产品还没有被头部清晰占位。

5. 对长期班 / 高客单家庭方案有什么启发
   - 长期班不要做“每周一节AI课”，而要做“季度主题制成长线”，比如品牌、餐饮、公益、城市发明四大主题，每季一个项目集和公开展示。
   - 高客单家庭方案可加入 `家庭共创+升学表达资产`：孩子作品集、英文 pitch 视频、家长复盘会、AI 使用规范、个性化项目孵化。这会比普通周课更接近高净值家庭的购买逻辑。
